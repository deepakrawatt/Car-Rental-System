# 🚗 Car Rental System

A **Car Rental System** application that allows users to browse available cars, book rentals, manage reservations, and process payments.  
The system is designed to streamline the car rental process for both customers and administrators.

---

## 📋 Features

### **For Customers**
- View available cars with details (model, price, features).
- Search and filter cars by type, brand, or price.
- Book a car for specific dates.
- View and manage bookings.
- Cancel or modify reservations.

### **For Admin**
- Add, update, or delete cars.
- Manage rental prices and availability.
- View all bookings and customer details.
- Generate rental reports.
